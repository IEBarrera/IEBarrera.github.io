# About me

I'm **IE Barrera**. I write code for the pleasure of it and, when needed, on
commission: I have built custom software for whoever needs it —including
this [moiréGL](/posts/moiregl/), made for an artist who works with moiré
patterns.

By training I'm a **chemist**, about to graduate with a thesis in theoretical
chemistry. Along the way I became a generalist: data handling, 3D graphics
(Houdini included), web development and everything that falls in between.

I'm interested in the odd crossings: where science, code and the visual
touch.
